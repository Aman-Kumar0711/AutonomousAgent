# Autonomous Lead Generation Agent

An end-to-end system that finds local businesses, audits their websites for issues, sends personalized cold outreach emails with a link to a beautiful audit report, and helps you convert them into paying clients.

## How It Works

```
┌─────────────┐    ┌──────────────┐    ┌────────────┐    ┌──────────────┐
│  1. SCRAPE   │───▶│  2. AUDIT    │───▶│  3. EXPORT │───▶│  4. OUTREACH │
│ Google Maps  │    │  Websites    │    │  Dashboard │    │  Send Emails │
│ + Search     │    │  20+ checks  │    │  to Vercel │    │  w/ report   │
└─────────────┘    └──────────────┘    └────────────┘    └──────────────┘
```

1. **Scrape** - Find businesses on Google Maps by type and location
2. **Audit** - Analyze each website for SEO, speed, security, and UX issues
3. **Export** - Generate data for the portfolio dashboard
4. **Outreach** - Send personalized emails with a link to their free audit report
5. **Follow-up** - Automated follow-up sequences (2 follow-ups)

The audit report lives on your Vercel-deployed dashboard — a beautiful, personalized page that shows each business exactly what's wrong with their site and how you can fix it. This makes your cold email almost impossible to ignore.

## Quick Start

### Prerequisites

- Python 3.11+
- Node.js 18+
- A [SerpAPI](https://serpapi.com) account (100 free searches/month)
- Gmail or [Resend](https://resend.com) for sending emails

### 1. Set Up Environment

```bash
# Clone and enter the project
cd AutonomousAgent

# Copy and fill in your configuration
cp .env.example .env
# Edit .env with your API keys and info

# Install Python dependencies
pip install -r agent/requirements.txt

# Install dashboard dependencies
cd dashboard && npm install && cd ..
```

### 2. Run the Agent

```bash
# Full pipeline: scrape → audit → export → outreach
python -m agent run --type "restaurants" --city "Austin" --state "TX" --limit 10

# Or run each step individually:
python -m agent scrape --type "dentists" --city "Denver" --state "CO" --limit 20
python -m agent audit --all
python -m agent export
python -m agent outreach --all --dry-run   # Preview first
python -m agent outreach --all             # Send for real
python -m agent follow-up                  # Send follow-ups
python -m agent stats                      # View statistics
```

### 3. Deploy the Dashboard

```bash
cd dashboard
npm run build    # Builds static site to /out
npx vercel       # Deploy to Vercel (follow prompts)
```

After deploying, update `PORTFOLIO_BASE_URL` in your `.env` file with the Vercel URL.

## Website Audit Checks

The analyzer runs **20+ checks** across these categories:

| Category | Checks |
|----------|--------|
| **Security** | SSL/HTTPS, Content-Security-Policy |
| **SEO** | Title tag, meta description, H1, alt tags, sitemap, robots.txt, Open Graph, structured data, canonical, heading hierarchy |
| **Performance** | Page speed (Google PageSpeed API), load time, GZIP compression |
| **Mobile** | Viewport tag, responsive design, media queries |
| **Marketing** | Analytics tracking, social media links |
| **UX** | Contact form, chat widget, favicon |
| **Accessibility** | Language attribute, skip navigation |

Each issue includes:
- **Impact level** (high / medium / low)
- **Business-specific impact** ("You're losing 20-30% of potential clicks...")
- **Actionable recommendation**

## Email Sequences

The system sends a 3-email sequence:

1. **Initial Outreach** (Day 0) — Personalized email referencing their Google Maps data, top 3 issues with business impact, link to free audit report, CTA to book a call
2. **Follow-up 1** (Day 3) — Highlights one critical issue with dollar impact, re-links the audit report
3. **Follow-up 2** (Day 7) — Final touch with social proof, last chance angle

All emails are:
- Mobile-responsive HTML with inline CSS
- CAN-SPAM compliant (unsubscribe option)
- Rate-limited (20/hour max to avoid spam flags)
- Professional design with branded accent colors

## API Keys Setup

### SerpAPI (Required)
1. Go to [serpapi.com](https://serpapi.com)
2. Sign up for a free account (100 searches/month)
3. Copy your API key to `SERPAPI_KEY` in `.env`

### Gmail SMTP (Option A)
1. Enable 2FA on your Google account
2. Go to [App Passwords](https://myaccount.google.com/apppasswords)
3. Generate an app password for "Mail"
4. Use your email as `SMTP_USER` and the app password as `SMTP_PASSWORD`

### Resend (Option B — Recommended)
1. Go to [resend.com](https://resend.com)
2. Sign up (free: 3,000 emails/month)
3. Add your domain and verify it
4. Copy API key to `RESEND_API_KEY` in `.env`

## Tips for Maximum Conversions

1. **Target small/local businesses** — They need help the most and are most likely to respond
2. **Start with restaurants and salons** — High volume, usually poor websites
3. **Personalize the CTA section** — Update `dashboard/components/CTASection.tsx` with your real contact info and portfolio
4. **Deploy before emailing** — Make sure your Vercel dashboard is live before sending outreach
5. **Follow up** — Most replies come from follow-up 1 or 2, not the initial email
6. **Build industry demos** — Use the domain grouping to create before/after demos for each industry
7. **Track results** — Run `python -m agent stats` regularly to monitor your pipeline
